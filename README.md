**To run the application:**
>pip install -r requirements.txt

>python gui.py

Note: need Ghostscript downloaded to run successfully

https://ghostscript.com/releases/gsdnld.html

Windows: 
>choco install ghostscript

Mac: 
>brew install ghostscript
